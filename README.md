# Machine Learning Studies

A repository dedicated to studying **Machine Learning**, containing notebooks, exercises, and final projects developed throughout the learning process.

## Repository Structure

The repository is organized into three main sections:

### 1. Python

Introduction and practice with Python for data analysis and machine learning.

Contents:

* Python fundamentals
* Data manipulation
* Exercises and notebooks
* Final project

### 2. Machine Learning Fundamentals

Core theoretical concepts that support machine learning models.

Contents:

* Basic ML concepts
* Data preprocessing
* Model evaluation
* Final project

### 3. Machine Learning

Implementation and experimentation with machine learning algorithms.

Contents:

* Model training
* Supervised learning algorithms
* Practical experiments
* Final project

## Technologies Used

* Python
* Jupyter Notebook
* NumPy
* Pandas
* Scikit-learn
* Matplotlib / Seaborn

## Objective

This repository was created as a **learning space to practice machine learning concepts**, experiment with algorithms, and document the development of practical projects.
